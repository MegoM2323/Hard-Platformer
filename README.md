## 📌 Описание

**Hard Platformer** — это 2D-платформер, созданный на Unity. В проекте реализованы:

- уровни со сценами и меню
- анимации игрока и врагов
- звуковые эффекты
- виртуальный джойстик
- настраиваемая физика
- поддержка Android-сборки
